🧾 Meta Evidence Collector

Graph API thường không trả đủ nguồn quảng cáo cho lịch sử cũ. Module meta-browser-sync dùng Playwright để mở Business Suite Inbox, đọc hội thoại và ghi bằng chứng vào Supabase.

Cách chạy trong Codespaces/VPS/local

cd meta-browser-sync cp .env.example .env # điền SUPABASE_URL và SUPABASE_SERVICE_ROLE_KEY npm install npx playwright install chromium npm run login npm run sync

Sau khi sync xong, quay lại /lead-tracker.

SQL cần chạy

Chạy file database/SUPABASE_PATCH_V6_1_META_EVIDENCE.sql trong Supabase SQL Editor.
