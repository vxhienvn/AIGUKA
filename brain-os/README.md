# AIGUKA Brain OS

Từ 5.3.1, Constitution/Rules/Policies/Casebook được tách thành module tri thức riêng.
Mục tiêu: Core ổn định, tri thức cập nhật bằng file lẻ.

- constitution/articles: điều khoản bất biến hoặc ưu tiên cao.
- rules: quy trình bán hàng showroom.
- policies: chính sách có thể thay đổi.
- casebook: tình huống thực tế dùng để đào tạo Brain.
