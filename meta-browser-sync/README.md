# AIGUKA Meta Browser Sync

Module này chạy trên máy/VPS có Chromium, không chạy trên Cloudflare Worker.

Bản V6.1 Stable ưu tiên Lead Tracker quét từ bảng `messages`. Browser Sync dùng cho bước nâng cao: đọc Business Suite để bổ sung đúng nguồn quảng cáo, link hội thoại và bằng chứng giao diện.

```bash
cd meta-browser-sync
npm install
npm run login
npm run sync
```
